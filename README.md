# b-challenge-android demo app
## Overview
Follows clean architecture guidelines (presentation, domain, data) with
MVMM architectural design pattern.

## Features used
- Material3 (Light theme and dark theme to be implemented)
- Jetpack Compose
- Retrofit2
- Dagger + Hilt
- Coroutines Flow
